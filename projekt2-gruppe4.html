Requirements for tecnical part:

A full CI/CD pipeline must be set up using the following:

  • Github actions as the “build server”
  • Azure web app for your Spring Boot Backend, set up to use your MySQL server
  • A VM (with Ubuntu) with Docker installed and a MySQL server running in a container
  • A Static WebServer hosting your Client-Side Code
  • With custom domain set up

Requirements for the Programming Part

  • A REST API, implemented according to the product owner’s wishes, using JPA with Hibernate and Spring Data JPA for database access
  • Test cases for relevant parts of the code
  • A separate pure JavaScript-based client, using the REST-API above for data access

What to include in the hand-in

  • Link to a short video (max 10 min.)
  • Link to the backend repository
  • Link to the frontend repository
  • Link to your deployed backend
  • Link to your deployed frontend
  • A short text explaining how far you came with the project.
  • A short reflection from the group reflecting on the educational/professional value of the last three project weeks
